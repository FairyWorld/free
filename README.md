>ps 重置插件使用方法
>- 1.启动你的IDE，如果上来就需要注册，选择：试用（Evaluate for free）进入IDE
>- 2.在编辑器启动界面, 将 ide-eval-resetter-1.0.4.jar 拖放到编辑器里, 提示 "RESTART" 重新启动.
>- 3.点击 "RESTART" 重启编辑器. 提示 ide-eval-resetter 已安装.
